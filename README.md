## url
- [www.wuqihong.cn](www.wuqihong.cn)

## build a local version of qihong`s blog
1. pull this code to your pc
2. ```cd qh-blog```
3. ```npm install```
4. ```hexo clean```
5. ```hexo generate```
6. ```hexo server```
7. check localhost:4000 to see qihong`s blog!
