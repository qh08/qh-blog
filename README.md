## url
- [www.wuqihong.cn](www.wuqihong.cn)

## build a local version of qihong`s blog
1. pull this code to your pc
2. ```cd qh-blog```
3. ```npm install```
4. ```npm install hexo -g```
5. ```hexo clean```
6. ```hexo generate```
7. ```hexo server```
8. check localhost:4000 to see qihong`s blog!
